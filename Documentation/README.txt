****************** A RAGMAN ******************
**********************************************

The Jar file "" launches the program. It must be placed in a directory with the file "anagrams.properties" and with a file containing the words to compare, its name must be specified into "anagrams.properties". 

A test suite has been implemented and is authomatically executed at every push/pull request by TravisCI, which is integrated with repository.

To run a jar file double click on it, the results of the elaboration is provided in the proper log file, "anagrams.log". Log files will be created into "log" folder; These one, if not existing, will be created by the program. 

To execute the jars Java 8 is required.
