****************** A RAGMAN ******************
**********************************************

The Jar file "AnagramFinder.jar" launches the program. It requires the files "anagrams.properties" and the one containing the words to compare; name of the latter must be specified into "anagrams.properties".

A test suite has been implemented and is automatically executed at every push request by TravisCI, which is integrated with repository.

To run a jar file double click on it, the results of the elaboration is provided in the proper log file, "anagrams.log". The log files are placed into the "log" folder; These one, if not existing, is created by the program. 

To execute the jars Java 8 is required.
